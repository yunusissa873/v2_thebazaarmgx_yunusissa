# Project Summary
The Bazaar is a Progressive Web App (PWA) marketplace designed to enhance the e-commerce experience by connecting buyers with verified vendors. It features user management, a comprehensive product catalog, integrated payment processing, and a modern interface. Recent developments have focused on improving core functionalities, including vendor tools, payment integrations, and analytics capabilities, to ensure operational success. Notably, the application now includes dynamic routing for individual vendor profiles.

# Project Module Description
The Bazaar consists of the following functional modules:
- **User Management**: Manages user profiles, vendor accounts, and staff roles.
- **Product Catalog**: Organizes product listings, variants, and categories.
- **E-commerce Operations**: Handles order processing, payments, and cart features.
- **Engagement Features**: Supports user reviews, coupons, and loyalty programs.
- **Communication Tools**: Facilitates messaging and chat between users.
- **Business Management**: Oversees vendor subscriptions and collaborations.
- **Build Badge**: Displays build information including version, build time, mode, and PWA status.
- **Navbar**: Enhanced navigation bar with improved layout and additional wishlist icon.

# File Description Inventory
- **README.md**: Project overview and setup instructions.
- **package.json**: Project dependencies and scripts.
- **tsconfig.json**: TypeScript configuration settings.
- **vite.config.ts**: Configuration for Vite build tool, including PWA settings.
- **supabase/schema.sql**: Database schema definitions.
- **src/components/ui/**: UI components like buttons and dialogs.
- **src/components/marketplace/**: Components specific to marketplace functionality, such as carousels and product cards.
- **src/components/shared/BuildBadge.tsx**: Displays build metadata (version, build time, mode, PWA status).
- **src/components/shared/Navbar.tsx**: Navigation bar with links, search bar, and icons.
- **src/data/products.json**: Mock data for products to be used in the Vendor Dashboard.
- **docs/audit/**: Contains audit reports and technical debt registers, including the consolidated audit report.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase
- **Build Tool**: Vite
- **Linter**: ESLint

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the project:
   ```bash
   pnpm run dev
   ```
