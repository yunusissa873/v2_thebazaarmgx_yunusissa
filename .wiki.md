# Project Summary
The Bazaar is a Progressive Web App (PWA) marketplace designed to provide an efficient e-commerce experience by connecting buyers with verified vendors. It features user management, a comprehensive product catalog, integrated payment processing, and a visually engaging interface inspired by Netflix, complete with a dark theme and smooth animations.

# Project Module Description
The Bazaar consists of the following functional modules:
- **User Management**: Handles user profiles, vendor accounts, and staff roles.
- **Product Catalog**: Manages products, variants, and categories.
- **E-commerce Operations**: Facilitates orders, payments, and cart functionalities.
- **Engagement Features**: Provides reviews, coupons, and loyalty programs.
- **Communication Tools**: Supports messaging and chat between users.
- **Business Management**: Manages vendor subscriptions and collaborations.
- **Build Badge**: Displays build information including version, build time, mode, and PWA status.

# File Description Inventory
- **README.md**: Project overview and setup instructions.
- **package.json**: Project dependencies and scripts.
- **tsconfig.json**: TypeScript configuration settings.
- **vite.config.ts**: Configuration for Vite build tool, including PWA settings.
- **supabase/schema.sql**: Database schema definitions.
- **src/components/ui/**: UI components like buttons and dialogs.
- **src/components/marketplace/**: Components specific to marketplace functionality, such as carousels and product cards.
- **src/components/shared/BuildBadge.tsx**: Displays build metadata (version, build time, mode, PWA status).

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase
- **Build Tool**: Vite
- **Linter**: ESLint

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the project:
   ```bash
   pnpm run dev
   ```
